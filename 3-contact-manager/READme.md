# Contact Manager

A full-stack MERN application for managing professional and personal contacts. This project features a unique "Midnight Noir" vintage aesthetic, robust server-side validation, dynamic sorting, and a split-screen layout.

## Features

* **Create Contacts:** Add new contacts with Name, Email, Phone, and Type (Personal/Professional).
* **View & Sort:** View all contacts in a scrollable list. Server-side sorting by Name, Email, or Date.
* **Delete:** Remove contacts with a confirmation check.
* **Validation:**
    * **Frontend:** Real-time feedback using Form Validation.
    * **Backend:** Secure data integrity using `express-validator`.
* **Vintage UI:** A custom "Midnight Library" theme with typewriter fonts, antique gold accents, and film-grain textures.
* **Responsive Layout:** Split-screen design (Form on the left, List on the right) that adapts to mobile devices.

## Tech Stack

* **Frontend:** React (Vite), Vanilla CSS (Custom Variables & Grid/Flexbox).
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB (Mongoose).
* **Tools:** Axios, Express Validator, CORS, Dotenv.

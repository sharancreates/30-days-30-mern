# Random Quote Generator 

A full-stack web application that fetches inspiring quotes from an external API and allows users to curate their own collection of favorites. 
Built with the **MERN** stack (MongoDB, Express, React, Node.js).

## Features

- **Random Quote Fetching**: Instantly retrieves quotes from `dummyjson.com`.
- **Save to Favorites**: Persist your favorite quotes to a MongoDB database.
- **Favorites Collection**: A dedicated page to view all saved quotes.
- **Delete Functionality**: Remove quotes you no longer want from your collection.
- **Responsive Design**: Clean and simple UI that works on desktop and mobile.

## Tech Stack

**Frontend:**
- React (Vite)
- React Router (Navigation)
- Axios (API Requests)
- CSS3 (Custom Styling)

**Backend:**
- Node.js & Express.js
- MongoDB & Mongoose (Database)
- Dotenv (Environment Variables)
- Cors (Cross-Origin Resource Sharing)

# Retro MERN Todo List

A full-stack To-Do List application built with the **MERN Stack** (MongoDB, Express, React, Node.js). 

<img width="859" height="630" alt="image" src="https://github.com/user-attachments/assets/433a947a-2c7e-4fa9-ae68-79b74c4b1fa0" />

It features a **Windows 95-inspired UI**, utilizing pure CSS to create 3D bevels, distinct borders, and that classic mechanical clicky feel.

## Tech Stack

* **Frontend:** React (Vite), Axios, Pure CSS (Retro Theme)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
